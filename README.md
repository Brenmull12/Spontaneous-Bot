# General Information
**A repository for Spontaneous Bot, a Discord bot developed for the Spontaneous Hangout Discord server.**

--

This bot runs using node.js and discord.js. For more information, please refer to discord.js.org.

For bug reports and feature requests, please submit an issue report on Github.

Contributions from any users not affiliated with Spontaneous Hangout will be ignored.
